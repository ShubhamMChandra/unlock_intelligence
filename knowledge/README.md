# Unlock Intelligence — Team Knowledge Base

A shared repository of learnings, strategies, playbooks, and artifacts from building Unlock Intelligence together.

## How to Use

- **Read before sessions** — Skim relevant docs before working sessions with Claude or team planning
- **Add after sessions** — After a productive session, capture what you learned in `learnings/`
- **Reference in decisions** — Link to strategy docs when making positioning or messaging choices
- **Claude reads this too** — Claude is configured to reference this folder, so insights here inform future AI-assisted work

## Structure

| Folder | What Goes Here | When to Add |
|--------|---------------|-------------|
| `learnings/` | Insights from working sessions, reviews, experiments | After any session that produced non-obvious insights |
| `strategy/` | ICP, buyer personas, GTM, competitive positioning | When strategy decisions are made or refined |
| `playbooks/` | Reusable processes and frameworks | When you find a process worth repeating |
| `ideas/` | Parking lot for future features, experiments, content | Anytime — low bar, just capture it |
| `artifacts/` | Approved copy, messaging frameworks, templates | When copy/messaging is finalized |

## Contributing

1. Use the template in `learnings/_template.md` for new learning entries
2. Date your entries (YYYY-MM-DD format)
3. Lead with the insight, then the context — busy readers skim
4. Tag what's actionable vs. what's just good to know
